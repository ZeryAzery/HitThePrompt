| Nom de l’attribut dans le schéma | Nom dans la console Active Directory | Description |
|----------------------------------|--------------------------------------|------------|
| sAMAccountName | Nom d’ouverture de session de l’utilisateur | Valeur que devra utiliser l’objet pour s’authentifier sur le domaine |
| UserPrincipalName | Nom d’ouverture de session de l’utilisateur + @domaine | Nom complet de l’utilisateur avec le domaine inclus (UPN) |
| description | Description | Description de l’objet |
| mail | Adresse de messagerie | Adresse de messagerie attribuée à l’objet |
| adminCount | — | Égal à 1 si compte administrateur protégé, sinon 0 |
| DisplayName | Nom complet | Nom complet affiché pour l’utilisateur |
| givenName | Prénom | Prénom de l’utilisateur |
| logonCount | — | Nombre d’ouvertures de session effectuées |
| accountExpires | Date d’expiration du compte | Date à laquelle le compte devient inutilisable |
| ObjectSID | — | Identifiant de sécurité unique de l’objet |
| pwdLastSet | — | Date de dernière modification du mot de passe |
| userAccountControl | — | État du compte (ensemble de flags binaires) |
